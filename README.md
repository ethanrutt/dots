# Ethan's dots

- heavily inspired from TechDufus's [dotfiles](https://github.com/TechDufus/dotfiles)

## Usage

- get ansible with pip
```sh
pip install ansible
```
- get ansible with package manager
```sh
sudo pacman -S ansible
```
- run the playbook
```
ansible-playbook main.yml
```
